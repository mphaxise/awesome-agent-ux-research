# Contributing

This repo is a curated docs collection, not a general dumping ground.

## Add sources that are at least one of these

- a primary-source product page
- a paper, benchmark, or design guideline
- a public discussion thread with real signal
- a GitHub repo that is directly relevant to agent UX, human-AI interaction, or agent evaluation

## Avoid adding

- low-signal social posts
- generic AI tooling with no UX connection
- repetitive "awesome" repos that do not add a new angle
- speculative commentary without links

## Suggested entry format

Use this shape when you add an item:

```md
- [Name](https://example.com/)
  - What it is: ...
  - Why it matters: ...
  - Category: ...
```

## Evidence standard

If a claim is interpretive, label it clearly as:

- `Evidence-backed`
- `Inference`
- `Assumption`

## Good additions

- concrete examples of agent failure modes
- public experiments comparing synthetic vs human evaluation
- design guidance for trust, override, explainability, and handoff
- tools that make agent UX evaluation more practical
